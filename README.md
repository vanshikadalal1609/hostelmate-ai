# 🏠 HostelMate AI

## 📌 Overview
HostelMate AI is an AI-powered student accommodation platform that helps students discover suitable PGs and rental accommodations near their college.

The platform uses machine learning to provide personalized recommendations based on user preferences and aims to simplify the process of finding a place to stay.

---

## 🚀 Features

### 🔍 Smart PG Finder
- Search PGs based on:
  - Budget
  - Distance from college
  - Food availability
  - WiFi
  - AC facilities

### 🤖 AI Recommendation System
Uses the K-Nearest Neighbors (KNN) algorithm to recommend the most suitable PGs.

### ⭐ Reviews and Ratings
- Student reviews
- Ratings for PGs
- Security information

### ❤️ Save PGs
Students can save their favorite PGs.

### 🤝 Future Feature: Roommate Matching
Find compatible roommates using machine learning.

HostelMate AI aims to solve these problems through an AI-powered recommendation platform.
